# Engenheiro de IA

Curso completo e gratuito — do Python do zero até agentes de IA avançados.

## Acesse o curso

🌐 **[engenheiro-de-ia.github.io](https://patrickcavalcante86.github.io/engenheiro-de-ia)**

## O que você vai aprender

| Fase | Conteúdo |
|---|---|
| **Fase 1** | Lógica de programação, Python completo, Git, Engenharia de Prompts |
| **Fase 2** | Como LLMs funcionam, Hugging Face, LLMs alternativas, API da OpenAI |
| **Fase 3** | FastAPI, Imagens/Áudio com IA, LangChain, RAG, Agno, MCPs, CrewAI |
| **Fase 4** | CursorAI, Claude Code, n8n, Deploy de servidores, Agentes no WhatsApp |

## Rodando localmente

```bash
pip install -r requirements.txt
mkdocs serve
```

Acesse em: `http://localhost:8000`

## Contribuindo

Pull requests são bem-vindos! Para contribuir:

1. Fork o repositório
2. Crie uma branch: `git checkout -b minha-contribuicao`
3. Faça as alterações nos arquivos em `docs/`
4. Abra um Pull Request
